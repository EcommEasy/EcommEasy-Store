# Ecomm-Store - Ecommerce Progressive Web Apps

Ecomm-Store is React and Node.js based eCommerce platform. 

Built with:
* Node.js v8.9
* React v16
* Redux
* Express
* Babel
* WebPack 4

## Store
Single-Page Application with React server-side rendering. 

### Requirements
* Node.js >= 8
